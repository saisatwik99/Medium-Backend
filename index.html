Add User

To add user you need to make a POST call to the following url :
https://medium-backend99.herokuapp.com/api/user


QUERY PARAMETERS

Field Type Description
name String Your Name
email String Your email

RESPONSE:
{
"followers": [],
"following": [],
"_id": "608e3c73f83420eddaa4a670",
"name": "Ravi Shankar",
"email": "ravishankar75@gmail.com",
"__v": 0
}

Create Article

To add user you need to make a POST call to the following url :
https://medium-backend99.herokuapp.com/api/article


QUERY PARAMETERS

Field Type Description
title String Title of the Article
text String Text
description String Description
author_id String User ID of the Author

Pass the query parameters as data (data: {}). You can also add image to article image(KEY) value(FILE). For more visit POSTMAN COLLECTION.
